# Neovim Config

- Lazy.nvim + plugins: Treesitter, LSP (clangd), nvim-cmp, gruvbox (hard), nvim-tree, lualine, autopairs, toggleterm, codeium  
- F5 — compile and run C++  
- Ctrl+n — toggle file tree  
- Ctrl+l — accept Codeium suggestion  
- Splits: <Leader>v (vertical), <Leader>s (horizontal)
